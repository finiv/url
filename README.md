Create a webpage prototype for URLs shortener.
Requirements
路 Create a form, that allows adding URL and system generates short unique URL:
- The format of generated URL: example.com/[hash];
- The short URL must be a valid URL;
- The URL must be shortened till 6 symbols hash, which contains alphanumeric symbols;
- Algorithm must recognize duplicate URL and instead of generating new short URL, show previously created;

路 After implementation, upon opening the short URL, the user must be redirected to the original URL.
路 Advantage, if functionality could work from folder (e.g.: example.com/something/[hash]).
路 For implementation use Laravel and Vue.js
